# Contributing to GH Branch Guard

Thank you for contributing! Please fork the repo, create a branch, run tests, and open a PR.

## Steps

1. `git clone ...`
2. `make install`
3. `make test`

## Code style
- Follow formatting/style guidelines (Black, ruff).
- Keep tests updated.

## Issue triage
- Add labels, add reproduction steps, link related issue/PRs.
